# Better Todos

## Re-create

Make a copy of `05-async-todos` and rename the app. Also update package name in `package.json`, update the `<h1>` in `App.tsx` and remove `<title>` from `index.html`.

## Installation

```bash
npm install
```

## Run

Start both client and server in separate terminals.

### Client

```bash
npm run dev
```

### Server

```bash
npm run server
```
