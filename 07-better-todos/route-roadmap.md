# Better Todos Route Roadmap

| url                 | description                   |
|---------------------|-------------------------------|
| `/`                 | Welcome page                  |
| `/todos`            | List of all todos             |
| `/todos/3`          | Show single todo ID 3         |
| `/todos/4`          | Show single todo ID 4         |
| `/todos/create`     | Show create todo form         |
| `/todos/3/edit`     | Show edit form for todo ID 3  |
